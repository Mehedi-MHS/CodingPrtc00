PreventDefault() vs StopPropagation()

Both are used to stop browsers default action(eg: click event). But has a little difference.

StopPropagation stops event for parent container only.ie: there is a container and it has button child.So if you use PreventDefault(), no action will be fired. And if you use StopPropagation() then button click will work but parent click will not work
